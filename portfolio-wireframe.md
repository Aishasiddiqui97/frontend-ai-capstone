## Layout Notes

| Section | Element | Purpose |
|---|---|---|
| Header | Logo + name + external links | Establish identity; provide quick access to GitHub/LinkedIn and resume download |
| Hero | Headline, proof statement, tech badges, CTA button | Hook within 5 seconds; confirm skill fit; drive primary action |
| Projects | 3 project cards with screenshots, tech tags, demo/repo links | Provide evidence; validate the proof statement |
| About | Short bio, current status, learning trajectory | Humanise; answer "who is this person?" |
| Contact | Simple form, direct email | Remove all friction from the primary action |
| Footer | Copyright, social links | Navigation safety net; professional anchor |

---

## Responsive Behaviour (Mobile)

- Navigation collapses to hamburger menu
- Hero text and CTA stack vertically
- Project cards stack to a single column
- Contact form expands to full width
